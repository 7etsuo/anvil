# Iron Soul Visual Style Bible v0.1 (Anvil module layer)

**Theme:** mythic agentic world — ancient knowledge meets emergent intelligence.

**Palette:** deep indigos, warm golds, ethereal blues, obsidian; subtle circuit glow.
NOTE: Pebble palette rulings do NOT apply here (different branch) — but if any MAP/Scoria-canon
subject is rendered, its ratified visual canon (Character Domain v1.0, VDF, Pebble Design
Ratification Addendum v1.0) WINS over this file.

**Camera:** high three-quarter top-down (isometric for the World Sim Map module).

**Lighting:** upper-left key, gentle rim on agents/entities.

**Principles:**
- One base image per identity → `image_edit` variants (idle, walk, attack, hit, death, skill)
- Readable silhouettes; subtle idle breathing; weighty movement
- Flat-background game-ready PNGs for easy keying; no text, no UI elements baked into sprites
- Greybox until real — declare paths first, assets fill in later

**Early subjects to define:**
- Player Agent
- Knowledge Sprites / Knowledge Nodes / Skill Entities
- Environment plates: Knowledge Sanctums, Training Grounds, Adversarial Arenas
- UI frames & icons: clean, glyph-based, knowledge-tree inspired

**Prompt template (Grok Imagine):**
```
High three-quarter top-down game sprite, [SUBJECT], mythic agentic style, deep indigo and warm
gold palette, subtle glowing circuit patterns, full body visible, solid flat background for
easy keying, consistent upper-left lighting, game-ready PNG, no text, no UI elements
```

here). Any asset, caption, or copy that could surface on a Planet Scoria Prime narrative
channel routes through reveal-warden first — no AI-explicit vocabulary reaches pre-reveal
narrative surfaces.
