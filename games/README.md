# Games

Titles in this directory consume Anvil; they do not define engine internals.
Reusable mechanics discovered while building a title belong in
[`../anvil/packages/`](../anvil/packages/).

| Game | Folder | Current status |
|------|--------|----------------|
| Gravewake | [`gravewake/`](./gravewake/) | Active, playable schema-v2 ARPG and reference consumer of `@anvil/authoring` + `@anvil/genre-arpg` |

Engine usage: [`../anvil/ENGINE.md`](../anvil/ENGINE.md). Gravewake usage and
controls: [`gravewake/README.md`](./gravewake/README.md).
