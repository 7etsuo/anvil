# Anvil

**Agent-native multi-genre game engine** for AI coding agents.

| Path | Role |
|------|------|
| [anvil/](./anvil/) | Engine / framework (active) |
| [games/](./games/) | Games that use Anvil |

## Docs

**Start here:** [`anvil/docs/design/README.md`](./anvil/docs/design/README.md)

## Quick start

```bash
cd anvil
pnpm install
pnpm -r run build
pnpm test
pnpm dev:hello
```
